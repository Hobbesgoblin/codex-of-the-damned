Cybelotron

{% endblock %} {% block archetype %}

Highlights

This is a special deck using {{ card("Cybele") }} and {{ card("Call the Great Beast") }}.

The idea is to put a Master: Discipline on {{ card("Cybele") }} to get her to 11 capacity, use {{ card("Magic of the Smith") }} to fetch a {{ card("Soul Gem of Etrius") }}, unlock with {{ card("Truth of a Thousand Lies") }}, then play {{ card("Call the Great Beast") }} with ORS and burn, triggering the {{ card("Soul Gem of Etrius") }}. A new {{ card("Cybele") }} comes back with the {{ card("Soul Gem of Etrius") }} and bleeds alongside the Great Beast.

This is a complicated combination but, if done right, {{ card("Cybele") }} and her "resurrection" can both bleed in the same turn for 3 each, plus 3 from the Great Beast. {{ card("Golconda: Inner Peace") }} is used on the Great Beast and {{ card("Redeem the Lost Soul") }} on the burnt {{ card("Cybele") }}, for a massive bloat.

{{ card("Dreams of the Sphinx") }} and {{ card("The Barrens") }} are used to discard the spurious {{ card("Magic of the Smith") }} once the combo is up and running.

The combat module is defensive, using only {{ card("Majesty") }}.

Tips & Tricks

{{ card("Antediluvian Awakening") }} can help burn an exhausted {{ card("Cybele") }} to avoid getting stuck in torpor.

Once {{ card("Cybele") }} is at 11 capacity with the {{ card("Soul Gem of Etrius") }}, {{ card("Truth of a Thousand Lies") }} can be used to get her bleed up to 5: If the prey plays {{ card("Archon Investigation") }}, a new {{ card("Cybele") }} just comes into play and bleeds again.

{{ card("Spying Mission") }} can be used if a bleed is bounced to avoid dealing damage to an ally.

Sadly, {{ card("Golconda: Inner Peace") }} can not be used on {{ card("Cybele") }}: as it does not burn her, it does not trigger the {{ card("Soul Gem of Etrius") }} effect.

Variants

There are only minor variations to this archetype. {{ card("Dark Influences") }} and {{ card("Unleash Hell's Fury") }} are often included.
