Spirit Marionette

{% endblock %} {% block archetype %}>

Highlights

This is a toolbox deck starring {{ card("Saulot, The Wanderer") }}. He uses {{ card("Spirit Marionette") }} to steal opponents vampires and take their blood with {{ card("Heidelberg Castle, Germany") }}.

The deck disciplines are BAT. {{ card("Saulot, The Wanderer") }} is accompanied by Tremere antitribu, and they can use {{ card("Infernal Pact") }} to be able to play B too.

A provides good intercept, wakes and bleed bounces with {{ card("Eyes of Argus") }}, {{ card("Telepathic Misdirection") }} and {{ card("My Enemy's Enemy") }}

T and B are used for combat, with {{ card("Anesthetic Touch") }} and {{ card("Theft of Vitae") }}, making for a nice "flight of fight" module.

The deck can slowly grind the whole table and use {{ card("Enkil Cog") }} and {{ card("Codex of the Edenic Groundskeepers") }} to lunge and destroy its successive preys.

Tips & Tricks

{{ card("Magic of the Smith") }} is used to fetch great equipments, depending on the table context. {{ card("Heart of Nizchetus") }} is the default choice.

{{ card("Renewed Vigor") }} is very strong combined with {{ card("Saulot, The Wanderer") }}'s ability, if any Tremere antitribu can use {{ card("Infernal Pact") }} to play it.

{{ card("Rutor's Hand") }} is a good way to improve {{ link("/strategy/fundamentals", _("A&B"), _anchor="AB") }}, provided one can pay the blood.

Variants

The Eurayle variant uses {{ card("Eurayle Gelasia Mylonas") }} as the star Salubri, as in {{ link("/deck-search", _("Yannick Gibert's deck"), twda_id="2010fncqlyon") }} from 2010.

An {{ link("/deck-search", _("older version from Tomasz Kowalewski"), twda_id="2k8crusadebreslau") }}, in 2008, do not even use the Tremere antitribu and their {{ card("Infernal Pact") }}, but low capacity A vampires instead.
