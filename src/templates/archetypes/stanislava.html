{% endblock %} {% block archetype %}>

Highlights

This is a stealth vote & bleed deck with {{ card("Stanislava") }} as a star. She uses {{ card("Govern the Unaligned") }} to help get other vampires out and {{ card("Villein") }} is used to get some pool back. The second vampire should be a Prince, preferrably {{ card("Ingrid Rossler") }}, to be able to play the numerous {{ card("Parity Shift") }}.

As the game progresses, more big capacity titled vampires get out and {{ card("Parity Shift") }} is used both to deliver pool damage and to bloat. Vampires begin to bleed, using j for stealth with {{ card("Form of Mist") }}, {{ card("Earth Control") }} and {{ card("Rapid Change") }}.

{{ card("Deflection") }} is used for bleed defence, with some {{ card("Sense the Savage Way") }} to unlock if need be.

Tips & Tricks

{{ card("Forced March") }} and {{ card("Instantaneous Transformation") }} are extremely versatile as they can be used both for stealth or additional action, allowing a vampire to bleed and call a vote in the same turn. {{ card("Stanislava") }} can use both, but some other vampires in the crypt can use one or the other.

{{ card("Murmur of the False Will") }} can be used both for bounces and bleed enhancement, depending on the context. However, they're not supposed to be used with {{ card("Stanislava") }} herself, as bleeding for more than 3 would expose her to {{ card("Archon Investigation") }}, a chance one would be wary to take with this deck.

A Prince is required, not only for the {{ card("Parity Shift") }}, but also for {{ card("Papillon") }}, an excellent device to bring some blood back on the vampires. {{ card("Jake Washington") }} is also a very efficient tool to that effect.

The {{ card("Seal of Veddartha") }} is an excellent equipment with this crypt, as most vampires miss either D or F. The +1 bleed bonus is nothing to cast aside either.

Variants

The archetype is very much known for Martin Weinmayer's build as he as been very successful with it and retained the top seat in the international classment for some years. The cards proportions may vary a little bit, some like to play some {{ card("Guard Dogs") }}, {{ card("Kine Resources Contested") }}, {{ card("Vele's Hunt") }} and/or {{ card("Claws of the Dead") }}. {{ card("Ecoterrorists") }} is also a common inclusion. Some alterations have been seen in the crypt too, like ditching {{ card("Rachel Brandywine") }} for an additional {{ card("Ingrid Rossler") }} or including {{ card("Iliana") }} or {{ card("Caitlin") }}, as they have excellent disciplines combination for a low pool, although no title to support the political side of the deck. {{ link("/deck-search", _("Janne LÃ¶nnqvist's deck"), twda_id="2016rfehf") }} from 2016 is a good example.

The Old School variant uses groups 1 and 2 in the crypt instead of the group 2-3 used here. It provides {{ card("Angus the Unruled") }} and {{ card("Don Cruez, The Idealist") }}, two justicars with a nice sets of disciplines. It allows for more Royalty cards like {{ card("Second Tradition: Domain") }} and {{ card("Fourth Tradition: The Accounting") }}. {{ link("/deck-search", _("Giulio De Cicco's deck"), twda_id="2014gtdtigi") }} from 2014 is a good example.
