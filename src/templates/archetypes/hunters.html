Highlights

This is a toolbox Imbued deck. They are mortal allies and follow specific rules. They use Conviction and Power cards.

The idea of the deck is to get some Imbued out, {{ card("Jennie") }} first if she's drawn, then equip them with weapons and armors to be able to hunt and take down vampires around the table. {{ card("Angel of Berlin") }} and {{ card("NRA PAC") }} are used to help equip them quickly.

As the game progresses, Powers make the Imbued more powerful. {{ card("Vigilance") }} is the most important, as it enhances {{ link("/strategy/fundamentals", _("A&B"), _anchor="AB") }} a lot. {{ card("Champion") }} and {{ card("Discern") }} help bump up the defence.

{{ card("Talbot's Chainsaw") }} is the key here, as its rush ability combined with its damage is very potent, and its disadvantage can be mitigated by {{ card("Rejuvenate") }} and {{ card("Jennie") }}'s ability

Finally, {{ card("Travis") }}, {{ card("The Church of Vindicated Faith") }} and {{ card("Inspire") }} are used to bloat and accelerate the crypt.

Tips & Tricks

{{ card("React with Conviction") }} is always the first Conviction an Imbued should fetch when he comes into play, as it protects them against anti-ally actions like {{ card("Entrancement") }}.

{{ card("Smiling Jack, The Anarch") }}, {{ card("Tension in the Ranks") }} and {{ card("Dragonbound") }} can improve pool damage a bit, but most damage come from bleed using {{ card("Strike with Conviction") }}. If the bleed action of {{ card("Strike with Conviction") }} is blocked, the Conviction is not lost, as its burn effect is the cost of the action, only payed if it is successul. It can then be used as a strike during the follow-up combat.

Variants

Decks in this archetype use different weapons and combat modules: the use of {{ card("Talbot's Chainsaw") }} is rare. {{ card("Target Vitals") }} is a common inclusion, and {{ card("Sniper Rifle") }} is a more common weapon, as in {{ link("/deck-search", _("Cory Busch Kendall's deck"), twda_id="2010reunionkamutco") }} from 2010.

The Paralyze variant uses the {{ card("Concealed Weapon") }} & {{ card("Flash Grenade") }} combination and has been played by {{ link("/deck-search", _("John Newquist in a 2008 NAC Qualifier"), twda_id="2k8originsnaq") }}.

The Apocalypse variant include more Gehenna events to cripple vampires around the table, as Imbued stay immuned to their effects. {{ link("/deck-search", _("Sergio Gracia's deck"), twda_id="2010famousims") }} from 2010 is a good example.
