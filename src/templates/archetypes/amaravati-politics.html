Amaravati Politics

{% endblock %} {% block archetype %}

Highlights

This is a breed & boon political deck using {{ card("Amaravati") }} as a star for his ability, then creating a swarm of Assamite with {{ card("The Embrace") }} and {{ card("Web of Knives Recruit") }}, bloating with {{ card("Consanguineous Boon") }} and {{ card("Haqim's Law: Leadership") }}.

The deck has a rather slow build-up but a high bloat that makes him sturdy. Furthermore, it delivers consistent pool damage with votes on one hand ({{ card("Reckless Agitation") }} and {{ card("Kine Resources Contested") }}) and bleed actions on the other hand (either naked bleeds or with the {{ card("Loss") }} action).

The deck has a strong stealth as O is one of his core disciplines.

Tips & Tricks

During the first half of the game, the deck should focus on breeding and bloating, surviving its predator. {{ card("Heart of Nizchetus") }} must come in play as early as possible, in order to cycle the aggressive actions foe the time being, and {{ card("Information Highway") }} can speed up this first phase.

Once a dozen Assamite are out, aggressive votes are used to destroy the prey's pool. <{{ card("The Sargon Fragment") }} can be used to fetch {{ card("Reckless Agitation") }} or {{ card("Loss") }} back from the ash heap.

{{ card("Amaravati") }} should be well protected, using {{ card("Yoruba Shrine") }} and {{ card("Direct Intervention") }} or sacrificing neonates to counter rush actions.

{{ card("Alamut") }} should be resplenished as much as possible. Vampires with o are supposed to get some bleed through (even for 1 pool) to get tokens on it fast.

Variants

Regarding the decklist example, some other cards can been included:

  • {{ card("The Uncoiling") }} to defend against {{ card("Scourge of the Enochians") }}, a common counter against breed decks like this one.
  • {{ card("The Khabar: Community") }} to help get the neonates bleed through.
  • {{ card("Perfectionist") }} and/or {{ card("The Coven") }} to help get some blood back on {{ card("Amaravati") }}.
  • Some {{ card("Deflection") }} and {{ card("Black Sunrise") }} to bump up the defence a bit.

    • There is a Laibon variant to this deck, using more copies of {{ card("Olugbenga") }} and {{ card("Kamau Jafari") }}, then replacing {{ card("The Embrace") }} by the cheaper {{ card("Bamba") }}. This also allows the use of some good Laibon cards like {{ card("Kduva's Mask") }} and {{ card("The Secret Must Be Kept") }}. {{ link("/deck-search", _("Riccardo Fabiani's deck"), twda_id="2011italyqtgenova") }} at the 2011 Italian Grand Prix is a good example.
