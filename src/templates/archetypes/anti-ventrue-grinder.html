Anti Ventrue Grinder

{% endblock %} {% block archetype %}

Highlights

Often spelled !Ventrue Grinder (the "!" prefix has been used to denote the antitribu clans for a while), this is a toolbox deck using the Ventrue antitribu clan and their fad disciplines combination.

The idea is to get the highest capacity vampire out, then use {{ card("Govern the Unaligned") }} at superior D to get more vampires out fast and cheap, then later to bleed with the help of {{ card("Conditioning") }} and {{ card("Bonding") }}.

The intercept module uses A for intercept, wakes and bounces, and d for additional bounces with {{ card("Deflection") }}.

The combat module uses f for damage prevention and {{ card("Weighted Walking Stick") }} to deal more damage.

The deck is on the slow side, grinding at the opponents blood and pool reserves during the game until it can finally lunge to oust its prey.

Variants

There are many small variations to this archetype:

  • {{ card("The Coven") }} is a rare inclusion: the deck is supposed to grind the blood out of the table, so giving opponents the opportunity to get some of it back is a bit of a stretch. {{ card("The Rack") }} is more often seen.
  • {{ card("The Kiss of Ra") }}, {{ card("Misdirection") }} and {{ card("Seduction") }} can be used to help get the bleed through.
  • {{ card("Abbot") }}, {{ card("Bowl of Convergence") }} and permanent intercept locations like {{ card("KRCG News Radio") }} are often included to help with intercept.
  • {{ card("Ivory Bow") }} and {{ card("Target Vitals") }} are often seen in the combat module.
  • {{ card("Freak Drive") }} is sometimes added to increase {{ link("/strategy/fundamentals", _("A&B"), _anchor="AB") }}.
