Highlights

This is a Vote deck using {{ card("Scout Youngwood") }} for her MORf disciplines and unique access to {{ card("Shattering Crescendo") }}, {{ card("Lily Prelude") }} and {{ card("Reckless Agitation") }}.

{{ card("Villein") }} and {{ card("Zillah's Valley") }} are used to get more vampires out fast and cheap.

The deck has serious stealth, with most vampires having O, and plays {{ card("Heart of the City") }} to enhance the bleed in the late game.

As all vampires except {{ card("Scout Youngwood") }} are Princes and Justicars, {{ card("Parity Shift") }} is heavily used both for pool damages and bloat.

Tips & Tricks

{{ card("Dmitra Ilyanova") }}'s ability is a great way to get more blood on the table for those {{ card("Villein") }} and {{ card("Reckless Agitation") }}.

With only 5 votes amoung its two star vampires, the deck can struggle to get the vote lock against other political constructs. Hence the inclusion of {{ card("Legendary Vampire") }}, {{ card("Bewitching Oration") }} and {{ card("Perfect Paragon") }} to get critical votes through. If the pool allows it and she is drawn, getting {{ card("Arishat") }} out is a good way to handle an Inner Circle, too.

Combat is purely defensive and uses only {{ card("Majesty") }}.

Variants

The archetype is seldom played and not many variations exist. Using a different crypt around can allow for different plays, for example using {{ card("Unmada") }} and {{ card("Benjamin Rose") }} for A and the excellent {{ card("Telepathic Misdirection") }} bounce. {{ link("/deck-search", _("Djordje Bjelajac's deck"), twda_id="2011ecfcq") }} from the 2011 EC FCQ is a good example of that.
