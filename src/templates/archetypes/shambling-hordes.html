Shambling Hordes

{% endblock %} {% block archetype %}>

Highlights

This is an allies rush deck using {{ card("Shambling Hordes") }}. The idea is to use {{ card("Liquidation") }} to put some {{ card("Shambling Hordes") }} in the ash heap, the recruit other ones, paying for it with the discarded ones. They then proceed to taking down opponents vampires while the vampires bleed.

Most Master cards are used to get some blood back on the vampires or cheapen the cost of {{ card("Shambling Hordes") }}.

As the deck has a solid rush potential, {{ card("Dragonbound") }} and {{ card("Fame") }} can help deliver more pool damage.

{{ card("Sudario Refraction") }} is used to get the cards discarded by {{ card("Liquidation") }} back into hand, providing a nice recursion for this deck.

Tips & Tricks

{{ card("Call of the Hungry Dead") }} and {{ card("Spectral Divination") }} help to bring out the {{ card("Shambling Hordes") }} without being blocked.

{{ card("Trap") }} can be used to take down big capacity vampires or allies with a defending ability like {{ card("Carlton Van Wyk") }}.

{{ card("The Unmasking") }} and {{ card("FBI Special Affairs Division") }} improve the allies quite a bit.

Variants

Many variants have a little bit more combat cards and do not bother with {{ card("The Parthenon") }}, like {{ link("/deck-search", _("Andrzej Jagla's deck"), twda_id="2013wbdpp") }} from 2013.

The Swedish variant uses {{ card("Tupdog") }} for additional minions in the ash heap, {{ card("Life in the City") }} to add blood to the vampires, and {{ card("Trochomancy") }} to help disrupt the recursion of decks using {{ card("Ashur Tablets") }}. {{ link("/deck-search", _("Otto Kukkasniemi's deck"), twda_id="2011snc") }} from the 2011 Swedish NC is a good example.
