Euro Brujah

{% endblock %} {% block archetype %}

Highlights

This is a Camarilla Royalty Toolbox Brujah deck using the few titled Brujah with d to be able to use {{ card("Govern the Unaligned") }} and some {{ card("Deflection") }}. Other than that, the deck uses the rather classical combination of {{ card("Second Tradition: Domain") }} and {{ card("Parity Shift") }} Camarilla Princes and Justicars have access to.

PC makes for a dreadful combat module, and {{ card("Theo Bell") }} and {{ card("Beast, The Leatherface of Detroit") }} have the ability to rush opponents vampires to take them down with it. Furthermore, {{ card("Second Tradition: Domain") }} is an excellent reaction that often results in a combat too.

{{ card("Govern the Unaligned") }} is used in the game beginning to get vampires out cheaper and faster, but it is an excellent bleed action later on.

Tips & Tricks

The oldest vampire with D should be put into play first in order to be able to use {{ card("Govern the Unaligned") }} on the younger vampires.

{{ card("Fame") }} and {{ card("New Carthage") }} are excellent pool damage enhancers.

The combat module has a lot of tricks to it: {{ card("Immortal Grapple") }} prevents Strike: Combat Ends and Dodge, {{ card("Pursuit") }} can be enough to get to close range against a single maneuver but, if the opponent insists on getting to long range, {{ card("Thrown Sewer Lid") }} can be used to take him down. {{ card("Blur") }} and {{ card("Pursuit") }} can deliver more damages through more strikes, then {{ card("Disarm") }} can put any vampire into torpor. Finally, {{ card("Sideslip") }} can help dodge a dreadful strike, get a {{ card("Disarm") }} through or prevent aggravated damage from {{ card("Wolf Claws") }} or {{ card("Chiropteran Marauder") }}.

Variants

There are a few variations around the crypt and the minion cards, especially the combat module. {{ card("Beast, The Leatherface of Detroit") }} is a rare inclusion and this example has a huge focus on combat. Other decks from this archetype can use d bleed enhancers like {{ card("Conditioning") }} and some {{ card("Bum's Rush") }}. {{ link("/deck-search", _("Michael Heyder's deck"), twda_id="2k8germanchamp") }} from the 2008 German NC is another example of the archetype.
