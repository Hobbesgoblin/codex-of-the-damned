Highlights

This is a wall deck starring {{ card("Saulot, The Wanderer") }} and {{ card("The Capuchin") }}, with some sidekicks having the AF disciplines

They all get Anarch using {{ card("Anarch Convert") }}, and that allows them to use good Anarch cards like {{ card("Anarch Revolt") }}, {{ card("The Anarch Free Press") }}, {{ card("Monkey Wrench") }} and {{ card("Guardian Vigil") }}.

The idea is to block problematic actions both from the prey and the predator, using A for intercept, weapons and F for damage prevention in combat. The deck slowly builds up a strong position and delivers some well-timed bleeds to the prey.

{{ card("Anarch Revolt") }} and {{ card("Smiling Jack, The Anarch") }} help improve pool damage, and {{ card("Monkey Wrench") }} can be used for the lunge.
