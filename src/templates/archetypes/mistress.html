Mistress

{% endblock %} {% block archetype %}>

Highlights

This is a stealth bleed & vote deck starring {{ card("Mistress Fanchon") }}. Her ability lets her fetch an {{ card("Alastor") }} to equip with an {{ card("Helicopter") }}, paying only 1 pool.

{{ card("Villein") }}, {{ card("Zillah's Valley") }} and {{ card("Information Highway") }} are used to get multiple big capacity vampires into play.

This version uses only the DO disciplines, a dreadful combination. {{ card("Obedience") }} and {{ card("Mental Maze") }} are used for combat defence.

Tips & Tricks

The key idea of this archetype is to abuse {{ card("Mistress Fanchon") }}'s ability to fetch the right card at the right time. The focus of the deck is to make sure she can do that and bleed every turn.

{{ card("Scourge of the Enochians") }} is included to cripple breed constructs, as they can be overwhelming for this slower deck.

Variants

The {{ card("Alastor") }} + {{ card("Helicopter") }} combination is not mandatory. Other methods can be found to improve {{ link("/strategy/fundamentals", _("A&B"), _anchor="AB") }}, for example using {{ card("Flurry of Action") }} like in {{ link("/deck-search", _("Benoît Moyen's deck"), twda_id="2010eternalvigilancelf") }} from 2010.

Danilo Torrisi played a {{ link("/deck-search", _("nice combination"), twda_id="2018bgpmb") }} in 2018: The {{ card("Might of the Camarilla") }} alongside {{ card("Possession") }} to get more Inner Circle into play fast and cheap.

Although this is a different archetype, {{ card("Mistress Fanchon") }} can be used for her AV combination, alongside {{ card("Malgorzata") }}, {{ card("Sha‑Ennu") }} or {{ card("Unmada") }} for a variation of the {{ link("/archetypes/tzimisce-wall", _("Tzimisce Wall")) }} archetype.
