Dmitri's Big Band

{% endblock %} {% block archetype %}

Highlights

This is a toolbox deck centered around the CAQ disciplines, with {{ card("Dmitri Borodin") }} as a star.

{{ card("Truth of Blood") }} and {{ card("Trochomancy") }} are used to bleed. {{ card("Disguised Weapon") }} along {{ card(".44 Magnum") }} and {{ card("Sniper Rifle") }} for combat. c provides some maneuverability with {{ card("Pursuit") }}.

A provides intercept and bounces, along the excellent {{ card("Black Sunrise") }} for additional unlocks. There is a bit of permanent intercept and {{ card("Smiling Jack, The Anarch") }} is supposed to be kept on the table to increase pool damage.

Tips & Tricks

{{ card("Enkil Cog") }} is an excellent bleed enhancer, but an {{ card("Archon Investigation") }} on the prey side is dangerous if it is used in conjunction with {{ card("Trochomancy") }}. Using {{ card("Enkil Cog") }} ability to bleed during the prey's turn prevents the use of {{ card("Archon Investigation") }} (if the bleed is not bounced): as it is an out-of-turn master, the prey can not play it during her own turn.

{{ card("The Fourth Cycle") }} is included as a solution against problematic Gehenna card but since {{ card("Anthelios, The Red Star") }} ban in 2016 it should not be included in the deck anymore. {{ card("The Unmasking") }} or a {{ card("Dragonbound") }} are not problematic and {{ card("Veil of Darkness") }} is too rarely played to be considered a threat.

Variants

There is no major variant for this archetype. The crypt can be altered a bit, {{ card("Infernal Pursuit") }} added to help with the rotation and {{ card("Target Vitals") }} for more damages. Instead or along the {{ card("Trochomancy") }}, other permanent bleed enhancers can also be added, like {{ card("J. S. Simmons, Esq.") }} and {{ card("Tasha Morgan") }}.
