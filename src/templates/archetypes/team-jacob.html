Highlights

This is a stealth & bleed deck using {{ card("Tunnel Runner") }}. An Akunanse is required to get the weremoles out, and {{ card("Jacob Fermor") }} is expected to come out and boost their bleed power. {{ card("Computer Hacking") }} is used to help them bleed harder, too.

The deck relies on additional vampires with d for {{ card("Deflection") }} bounces. As the deck does not block so much, this is essential for defence.

Tips & Tricks

As the deck is supposed to get an Akunanse, {{ card("Jacob Fermor") }} then a low capacity vampire with d, it has a bit of each in the crypt and plays a bunch of {{ card("Wider View") }} and an {{ card("Anarch Convert") }} to rotate through them if necessary.

{{ card("Tunnel Runner") }} are expensive allies. getting {{ card("Powerbase: Tshwane") }} and/or {{ card("Charisma") }} out fast is a lot of help.

{{ card("Predator's Transformation") }} provides additional stealth to get the allies into play without being blocked.

{{ card("Aid from Bats") }} is used as a defensive combat card. {{ card("Tunnel Runner") }} can use it, but most Akunanse also can.
