Highlights

This is a special deck using Mind Rape alongside a big capacity vampire with DO, Lucian here. The idea is to steal the opponents vampires and use Heidelberg Castle, Germany to deprive them of their blood and equipment.

Using Malkavian and Madness Network brings an edge to this idea, as Mind Rape can be played on the predator turn, to get control of the vampire in the immediately following turn, for example.

The deck uses Second Tradition: Domain for defence, together with Obedience and Mental Maze to avoid combat.

Bleed and votes are used to apply pool damage, in combination with a strong stealth module using O.

Tips & Tricks

Scourge of the Enochians is used as a counter against breed constructs that can be overwhelming for this deck.

Metro Underground lets a vampire bleed or vote, then unlock at the end of the turn, only to use Madness Network for another action during another Methusalah's turn.

Variants

There is a Ventrue variant using Arika as a star. Erol Hammer's deck from 2009 is a good example.

Errata

Pentex™ Subversion has been modified by an errata in September 2019. It does not prevent a vampire to act anymore, so it is less powerful and should probably not be included in the deck anymore.

Parity Shift has also been modified in the September 2019 errata. The amount of pool is no longer equal to the number of Methusalah, but is stable at 3. The card is then less powerful in the game beginning, and should maybe be included in less copies.

Temptation of Greater Power has been banned in 2016 and should be removed from the deck.
