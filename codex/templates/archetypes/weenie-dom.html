Highlights

This is a weenie bleed deck using the d discipline. The idea is to get as many low capacity vampires out as fast as possible, and bleed the preys one after the other as fast as possible.

{{ card("Govern the Unaligned") }} combined with {{ card("Conditioning") }}, {{ card("Seduction") }}, and {{ card("Command of the Beast") }} make for hard to block dreadful bleeds for 7.

As the game progresses, some D bounce can be used to add to the damages if the predator becomes aggressive.

The Master module uses {{ card("Dominate") }} to enhance the vampires, but also {{ card("The Parthenon") }} and {{ card("Ashur Tablets") }} for a bit of recursion. {{ card("Jake Washington") }}, {{ card("Life in the City") }} and {{ card("Giant's Blood") }} are used to bring back some blood on the vampires.

The combat modules uses only {{ card("Absorb the Mind") }} for defence. It is a bit weak, but d has nothing better to offer.
