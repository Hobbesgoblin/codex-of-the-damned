Council of Doom

{% endblock %} {% block archetype %}

Highlights

This is a stealth vote & bleed deck using Multi Master Phase Actions (MMPA). It uses {{ card("Huitzilopochtli") }}, {{ card("The Parthenon") }} and {{ card("Rumors of Gehenna") }} to get more Master actions, then {{ card("Villein") }} and {{ card("Information Highway") }} to get big capacity vampires into play fast and cheap.

{{ card("Ashur Tablets") }} and {{ card("Liquidation") }} are used to bloat and get some recursion to adapt the deck content to the table context for maximum efficiency.

The deck delivers bool damage from stealthed naked bleeds from the Inner Circle {{ card("Arika") }} and {{ card("Leandro") }}, and also damage-dealing votes.

{{ card("Majesty") }} is used for combat protection, {{ card("Deflection") }} to bounce bleeds and O for high stealth using {{ card("Into Thin Air") }} and {{ card("Elder Impersonation") }}.

Tips & Tricks

If drawn, getting {{ card("Huitzilopochtli") }} into play first for more Master actions is best.

This is an old archetype (hence the example deck name), and this version uses many cards since banned: {{ card("Anthelios, The Red Star") }}, {{ card("Temptation of Greater Power") }} and {{ card("Lilith's Blessing") }}. No doubt removing them makes the deck less efficient, and modifications would have to be made.

Variants

No variant of this deck exist per se, although one could argue that the {{ link("/archetypes/girls-will-find-inner-circle", _("Girls Will Find Inner Circle")) }} archetype is a more or less direct heir to this archetype.

Many minor variations on the crypt or minion cards exist. Including {{ card("Etrius") }}, {{ card("Obedience") }} for combat defence, {{ card("Zillah's Valley") }} for crypt acceleration, some d bleed enhancers like {{ card("Conditioning") }} or {{ card("Carlton Van Wyk") }} for defence, are quite usual. {{ link("/deck-search", _("Ruben Feldman's deck"), twda_id="2k9lvmq") }} from 2009 is a good example.
